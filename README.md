# Development Setup

## Prerequisites

- Node.js >= 22
- npm >= 10
- Git


## 1. Navigate to the Project

```bash
cd Anindya-workspace
```

---

## 4. Install Dependencies

```bash
npm install
```

This installs all packages listed in `package.json`.

---

## 5. Start the Development Server

```bash
npm run dev
```

---

## 6. Build the Project

```bash
npm run build
```

---

## 7. Preview Production Build

```bash
npm run preview
```

---

## Useful Commands

Run ESLint

```bash
npm run lint
```
